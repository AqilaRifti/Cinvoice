# EarningsChart Component

## Overview

The `EarningsChart` component displays investment performance over time using an area chart with gradient fill. It provides interactive time range selection and responsive design for the Investor dashboard.

**Validates: Requirements 4.2, 8.1, 8.2, 8.3, 8.4, 8.6, 8.8**

## Features

- ✅ Area chart with gradient fill
- ✅ Time range selector (7D, 30D, 90D, All)
- ✅ Interactive tooltip showing date and value
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Empty state for new investors
- ✅ Loading state with skeleton
- ✅ Smooth animations
- ✅ Theme-aware colors using CSS variables

## Usage

```tsx
import { EarningsChart } from '@/components/investor';

export default function InvestorDashboard() {
  return (
    <div className="space-y-6">
      <EarningsChart />
    </div>
  );
}
```

## Features Detail

### Time Range Selector

The component provides four time range options:
- **7D**: Last 7 days
- **30D**: Last 30 days (default)
- **90D**: Last 90 days
- **All**: All time (1 year)

The chart data automatically updates when the time range changes.

### Interactive Tooltip

Hovering over the chart displays a custom tooltip showing:
- Full date (e.g., "Jan 15, 2024")
- Earnings value with 2 decimal places
- Smooth fade-in animation

### Gradient Fill

The area chart uses a gradient fill that:
- Starts with 30% opacity at the top
- Fades to 0% opacity at the bottom
- Uses the theme's primary color
- Adapts to light/dark mode

### Statistics Display

The header shows:
- Current earnings value
- Percentage change from start of period
- Color-coded change indicator (green for positive, red for negative)

### Empty State

When no data is available, displays:
- Icon illustration
- "No Earnings Data" message
- Call-to-action text encouraging investment

### Responsive Design

- **Mobile**: Stacked header layout, full-width chart
- **Tablet**: Side-by-side header with time selector
- **Desktop**: Optimized spacing and larger chart area

## Integration with Real Data

Currently, the component uses mock data generated by `generateEarningsData()`. To integrate with real blockchain data:

1. Create a custom hook (e.g., `useEarningsData`) that:
   - Fetches investment history from the FinancingPool contract
   - Calculates cumulative earnings over time
   - Formats data into `EarningsDataPoint[]` format

2. Replace the mock data generation:

```tsx
// Instead of:
const chartData = useMemo(() => {
  return generateEarningsData(daysMap[timeRange]);
}, [timeRange]);

// Use:
const { data: chartData, isLoading } = useEarningsData(timeRange);
```

3. Example hook implementation:

```typescript

  const { address } = useAccount();
  
  return useQuery({
    queryKey: ['earnings', address, timeRange],
    queryFn: async () => {
      // Fetch investment events
      // Calculate earnings over time
      // Return formatted data
    },
    enabled: !!address,
  });
}
```

## Styling

The component uses:

## Accessibility

- Time range buttons have `aria-label` attributes
- Chart uses semantic colors with sufficient contrast
- Tooltip provides clear information on hover
- Keyboard navigation supported via ToggleGroup

## Performance

- Chart data is memoized to prevent unnecessary recalculations
- ResponsiveContainer handles chart resizing efficiently
- Animations use GPU-accelerated properties
- Debounced rendering (2000ms) prevents laggy behavior during animations

## Testing

To test the component:

```tsx
  });
});
```
